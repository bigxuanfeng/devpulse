## 今天做了什么

从零搭建了 DevPulse——开发者个人效能面板。

### 技术选型
- Next.js 16 + React 19 + TypeScript + Tailwind CSS v4
- Recharts 做图表、Zustand 做状态管理
- 深色/白天双主题，暖琥珀 accent

### 仪表盘
- 4 个 KPI 卡片：AI 成本、提交数、活跃天数、项目健康
- 代码活动热力图 + AI 成本趋势图
- 数据来源：DeepSeek API 余额 + 本地 Git 仓库解析

### 日记系统（花了最多时间）
- 原地展开编辑：点卡片展开成查看模式，点"编辑"进入编辑，保存后收回
- 标题 + 日期 + 正文，支持同一天多篇日记
- 侧栏：日期区间选择、标签/项目筛选、上下箭头排序、管理弹窗
- 搜索：标题/内容/标签/项目，Enter 触发不丢焦点
- 编辑区：项目和标签用输入框管理，不再从文本解析 @
- 存储：data/diary/{id}.md，YAML frontmatter

### 踩坑记录
- toISOString() 取的是 UTC 时间，凌晨时新建日记日期比实际少一天，改用 getFullYear/getMonth/getDate
- @dnd-kit 在 SSR 下有 hydration 问题，加了固定 id 后还是不够稳，最后换成上下箭头排序，反而更简洁
- Tailwind v4 的 @theme inline 不支持运行时切换，改用了 CSS 变量方案
- 连续快速创建日记时 Date.now().toString(36) 可能重复，加了随机后缀
